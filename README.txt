PROJECT : Lucro Analyzer (LAnalyzer)

Author : Åke Hansson Lucro Consulting AB

Getting Started : Download the project from GitHub:

https://github.com/akebhans/LAnalyzer

to your local repository.

Prerequisite:

To run locally : Start the project preferably in Visual Studio 2017.

Database: Need access to a SQL Server instance. The database LucroAnalyzer is created automatically when you start the application.

Edit the Web.config files in the <connectionStrings> section to set the connectionstring corresponding to your settings.

Purpose: Purpose of this project is to have an application where you can upload csv-type files with data easily and fast to analyze the data 
in these files with a stardardized UI.

A live version is avaiable at : http://lonsamhet.se
